# Quick preview

  • Download docker-compose.ymlopen in new window file
  • run docker-compose pull to be sure to have the last version of AKHQ
  • run docker-compose up
  • go to http://localhost:8080open in new window

It will start a Kafka node, a Zookeeper node, a Schema Registry, a Kafka Connect, fill with some sample data, start a consumer group and a kafka stream & start AKHQ.
